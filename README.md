# strong-zero-css-art

A subpar CSS art of a Strong Zero can.

I always wanted to make some sort of "CSS art" of something dumb.

## Commands

- Install dependencies - `npm i`
- Live reload - `npm run serve`
- Watch SCSS - `npm run watch`
